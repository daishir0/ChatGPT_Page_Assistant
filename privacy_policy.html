ChatGPT Text Assistant Privacy Policy

Information We Collect

This extension only processes the following information:

  • Text selected by users on web pages
  • Questions entered by users
  • Current page title and URL (used as context for questions)
  • User settings and preferences (stored locally)

How We Use Information

The collected information is used solely for:

  • Generating question prompts for ChatGPT
  • Providing extension functionality
  • Storing user preferences locally

Information Storage and Sharing

This extension:

  • Does not store user data on external servers
  • Does not share data with third parties
  • Only stores settings locally using Chrome's storage API
  • Does not track or monitor user behavior

ChatGPT Integration

This extension automates interaction with ChatGPT's web interface but is an independent third-party product not affiliated with OpenAI. Conversations on ChatGPT are subject to OpenAI's privacy policy.

Data Security

All data processing happens locally in your browser. No information is transmitted to external servers except when you choose to submit questions to ChatGPT.

User Control

Users can:

  • Customize prompt templates in settings
  • Enable or disable temporary chat mode
  • Uninstall the extension at any time to remove all stored data

Contact

For questions about this privacy policy, please contact us through the Chrome Web Store extension page.
